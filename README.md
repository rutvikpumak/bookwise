# Bookwise
  A video library website based on Books Reviews
</div>

## **How to install and run locally ?**

```
$ git clone https://github.com/rutvikpumak/bookwise.git
$ cd bookwise
$ npm install
$ npm start
```
## **Features -**

- Home page
- Video listing page
- Filter Videos
- Search video
- Individual video page
- Like video 
- Watch Later Video
- History video
- Create playlist
- Comment on video
- Authentication:
  - User Signup
  - User Login
 - Fully Responsive

## **Built with -**

- React JS
- React Context API + useReducer
- React Router v6
- Vanilla CSS integrated with BoostMat UI Component Library
